```python
@asynccontextmanager
async def lifespan(app: FastAPI) -> AsyncGenerator[None, None]:
    """Application lifespan management with startup validation.

    Per D-04: Enforce ENCRYPTION_KEY configuration at startup.
    Per D-05: Log startup security state for operational awareness.
    """
    try:
        logger.info(
            "Starting QueryGPT API",
            version=settings.APP_VERSION,
            environment=settings.ENVIRONMENT,
        )

        # Validate encryption key configuration (fails fast in production)
        settings.validate_secrets()

        # Log security posture
        if settings.is_using_default_secrets:
            logger.warning(
                "Using default encryption key",
                environment=settings.ENVIRONMENT,
                message="🔒 Running with default encryption key. This is acceptable for development only. "
                        "Change ENCRYPTION_KEY environment variable for production.",
            )
        else:
            logger.info(
                "Using explicit encryption key",
                environment=settings.ENVIRONMENT,
                key_length=len(settings.ENCRYPTION_KEY),
            )

        # Log other security-relevant settings
        logger.info(
            "Startup configuration",
            debug_mode=settings.DEBUG,
            database_driver=settings.DATABASE_URL.split(":")[0] if settings.DATABASE_URL else "unknown",
        )

        yield  # Application is running

    except ValueError as exc:
        # Configuration validation failed (e.g., production without ENCRYPTION_KEY)
        logger.critical(
            "Configuration validation failed - application cannot start",
            error=str(exc),
        )
        raise
    except Exception as exc:
        # Unexpected startup error
        logger.critical(
            "Unexpected error during startup",
            error=str(exc),
        )
        raise
    finally:
        # Shutdown (currently no cleanup needed, but pattern ready)
        logger.info("Shutting down QueryGPT API", version=settings.APP_VERSION)
```

**Key requirements:**
1. Call settings.validate_secrets() (which raises ValueError if config is invalid)
2. Let ValueError propagate (fail fast if production misconfigured)
3. Log startup security state: explicit key vs default key
4. Log other relevant settings (DEBUG mode, database driver)
5. Use logger.critical() for startup failures (highest severity)
6. Handle unexpected errors gracefully (log and propagate)
7. Log shutdown cleanly

The flow:
- validate_secrets() raises ValueError → caught by outer exception handler → logger.critical → app doesn't start
- validate_secrets() succeeds → log security state → yield → app runs
- Shutdown: finally block logs shutdown message
